I had spent months thinking about renovating my blog. Which framework? What theme?

I looked to other portfolios for inspirationâ€”and often left annoyed by constant animations and stock graphics. Just let me read the content! All it needs is a little bit of style.

You don't need frameworks. Featherweight blogs aren't nails you want to hit with those heavy hammers. If you're a dev, you don't need to flex them on your website; the actual flex is using the right tool for the job.

So I wrote this website to be simple, fast, and responsive by default. There are no animations, no fancy menus, and no backends. I'm proud to be using No-SQL for this one.

It's Web 1.0 bliss, and it's oddly refreshing.
